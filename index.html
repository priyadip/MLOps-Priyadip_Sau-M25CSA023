📊 Key Findings

🏗️ Architecture Choice

ResNet-18 outperforms or matches ResNet-50 on both datasets while training 2× faster. Deeper networks don't always mean better performance for simpler tasks.

⚙️ Optimizer Selection

SGD excels on MNIST (simpler patterns), while Adam is superior for FashionMNIST (complex features). Optimizer choice should be task-specific.

📈 Learning Rate Impact

Learning rate is the most influential hyperparameter. Wrong choices can cause 20%+ accuracy drops, especially on FashionMNIST.

🚀 GPU Acceleration

GPU training provides 12-24× speedup with negligible accuracy difference. Essential for efficient hyperparameter search.

🎯 SVM Competitiveness

SVM achieves 97.94% (MNIST) and 90.31% (FashionMNIST) — only 1.4–1.7% below deep learning, without GPU requirement.

📉 PCA Efficiency

PCA reduces training time by 12× while maintaining 97%+ accuracy on MNIST. Effective for SVM when time is constrained.

📝 Technical Notes

Configuration Details

All experiments used: USE_AMP=True (mixed-precision training), 70%-10%-20% train-val-test split, PyTorch with torchvision, and scikit-learn for SVM.

Model Storage

Best SVM models exceed 100MB due to large support vector counts. GitHub file size limits prevent direct storage, so models are hosted on Hugging Face.

Additional Resources

All Python source files (.py), training logs, and experimental outputs are stored in Google Drive for complete reproducibility.

⚡ ART vs Scratch FGSM

Both attack methods saturate at ~10% accuracy beyond ε=0.20, where perturbation magnitude overwhelms the original image signal entirely.

🔬 PGD vs BIM

PGD's uniform large-step perturbations create more detectable artefacts (99.93%) vs BIM's smaller iterative steps (99.48%).
